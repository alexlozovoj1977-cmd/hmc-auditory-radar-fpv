# data/

Тут планується розміщення:

- синтетичних аудіосценаріїв для тренажера (WAV/FLAC),
- за потреби — посилань/маніфестів на реальні польові записи FPV (якщо дозволяє безпека).

Структура (пропозиція):

- `synthetic/` — згенеровані сигнали (модель шум + гармоніки + HMC),
- `real/` — опис і метадані реальних записів (без публікації сирих файлів, якщо є обмеження).
