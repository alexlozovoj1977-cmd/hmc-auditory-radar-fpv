# src/

Тут планується код для:

- генерації FPV-подібних звукових сигналів (noise + harmonics + HMC),
- створення сценаріїв для тренажера (послідовності «прольотів»),
- базового CLI/GUI-програвача сценаріїв.

Мінімальна ідея:

- `generator/` — функції для синтезу сигналу FPV з параметрами (тип дрона, відстань, азимут, HMC-параметри),
- `scenarios/` — опис тренувальних сценаріїв (JSON/YAML),
- `player/` — простий плеєр, що відтворює сценарій через 4-канальний аудіовихід.
